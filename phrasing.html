Phrasing Elements tag สำหรับจัดรูปแบบ ข้อความ

ตัวอย่างการใช้ strong แสดง ตัวหนา:

สามารถใช้ จัดรูปแบบ ตัวเอียงแบบ em หรือ ตัวเอียงแบบ italic text for emphasis, และ ตัวหนา ใช้ strong หรือ ใช้ b สำหรับข้อความที่ต้องการเน้น.
และ สามารถใช้ mark เพื่อ highlight เพื่อเติมสี แบบ ไฮไลท์.

นี่คือตัวอย่างของ abbr CSS
tag code <h2>ตัวอย่างการใช้ code <h2> เพื่อแสดง inline code.
ถ้าต้องการ ใส่การอ้างอิง citation จะสามารถใช้ tag cite, use the Cite Tag แบบนี้.

สามารถใส่ small เพื่อแสดง ตัวเล็ก เพื่อการพิมพ์แบบ ละเอียด
สามารถใช้ sub เพื่อแสดง ตัวห้อย subscript และ sup เพื่อแสดง ตัวยกsuperscript
สำหรับ ตัวอักษรทางวิทยาศาตร์ ต่างๆ เช่น H2O or E=mc2.

การขีดเส้นใต้ ใช้ tag u underline เพื่อขีดเส้นใต้, และ del เพื่อขีดฆ่า deleted
หรือ tag s เพือขีดกลาง strikethrough เพื่อบอกว่า ขีดข้อความออก.
หรือ ใช้ ins เพื่อ inserted เพื่อบอกว่า เพิ่มข้อความเข้าไป.

ใช้ q เพื่อ quote tag แสดงเครื่องหมายคำพูด quote , หรือ a anchor tag เพื่อแสดง link.

                  
                    <h1>Phrasing Elements tag สำหรับจัดรูปแบบ ข้อความ</h1>
                    <p>ตัวอย่างการใช้ strong  <strong>แสดง ตัวหนา</strong>:</p>
            
                    <p>สามารถใช้ จัดรูปแบบ <em>ตัวเอียงแบบ em </em> หรือ <i> ตัวเอียงแบบ italic</i> text for emphasis,
                     และ ตัวหนา <strong>ใช้ strong</strong> หรือ <b>ใช้ b </b> สำหรับข้อความที่ต้องการเน้น. <br/>
                    และ สามารถใช้ <mark> mark เพื่อ highlight</mark> เพื่อเติมสี แบบ ไฮไลท์.</p>
            
                    <p>นี่คือตัวอย่างของ abbr <abbr title="Cascading Style Sheets">CSS</abbr><br/>
                     tag code <code>&lt;h2&gt;ตัวอย่างการใช้ code &lt;h2&gt;</code> เพื่อแสดง inline code. <br/>            
                    ถ้าต้องการ ใส่การอ้างอิง citation จะสามารถใช้ tag cite, use the <cite>Cite Tag</cite> แบบนี้.</p>
            
                    <p>สามารถใส่ small เพื่อแสดง <small>ตัวเล็ก</small> เพื่อการพิมพ์แบบ ละเอียด <br/>
                    สามารถใช้ sub เพื่อแสดง ตัวห้อย <sub>subscript</sub> และ sup เพื่อแสดง  ตัวยก<sup>superscript</sup><br/>
                    สำหรับ ตัวอักษรทางวิทยาศาตร์ ต่างๆ เช่น H<sub>2</sub>O or E=mc<sup>2</sup>.</p>
            
                    <p>การขีดเส้นใต้ ใช้ tag u <u>underline</u> เพื่อขีดเส้นใต้, และ del เพื่อขีดฆ่า <del>deleted</del> <br/>
                    หรือ tag s เพือขีดกลาง <s>strikethrough</s> เพื่อบอกว่า ขีดข้อความออก. <br/>
                    หรือ ใช้ ins เพื่อ <ins>inserted</ins> เพื่อบอกว่า เพิ่มข้อความเข้าไป.</p>
            
                    <p>ใช้ q เพื่อ <q>quote tag</q> แสดงเครื่องหมายคำพูด quote , 
                    หรือ a <a href="#">anchor tag</a> เพื่อแสดง link.</p>
